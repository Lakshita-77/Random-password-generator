# 🔐 Random Password Generator

A simple and beginner-friendly **Password Generator** built using **Python**. This project generates secure random passwords based on user preferences such as password length, numbers, and special characters.

## 📌 Features

✅ Generate random passwords  
✅ Choose minimum password length  
✅ Option to include numbers (0–9)  
✅ Option to include special characters (@, #, $, %, etc.)  
✅ Beginner-friendly Python project  

---

## 🛠️ Technologies Used

- Python
- Random Module
- String Module
